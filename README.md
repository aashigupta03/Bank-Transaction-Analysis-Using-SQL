# Bank-Transaction-Analysis-Using-SQL
This project analyzes bank transaction data using MySQL. The goal is to understand spending patterns, income trends, and category-wise expenses using SQL queries.

## Dataset

* Transaction records
* Deposits
* Withdrawals
* Account balance history

## SQL Concepts Used

* SELECT
* WHERE
* GROUP BY
* ORDER BY
* Aggregate Functions
* Subqueries
* Window Functions
* Case Statements

## Key Analyses

### Total Income and Expenses

Calculated overall deposits and withdrawals.

### Category-wise Spending Analysis

Identified categories with the highest spending.

### Monthly Spending Trends

Analyzed spending behavior over time.

### Ranking Categories

Used SQL window functions to rank categories based on expenditure.

### Customer Financial Label

Used case statements to identify saving or balanced customer

## Key Insights

* Identified major spending categories.
* Calculated net savings.
* Analyzed transaction frequency.
* Detected high-value transactions.
